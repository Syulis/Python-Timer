操作設定

#<実行キー>
#数値入力系はTabでカーソルを外してから実行してください
#クリックでも可(同時実行と終了のみショートカットなし)

$Timeout Save
時間切れ後のデータ保存実行の有無

$Start Count
タイマー開始時のサウンドの有無

$Robocon2019
ワンクリックでロボコン2019仕様に

$Mute
タイマー開始時、終了時のサウンドをミュート

$Normal Color
Robocon2019仕様ではないときのタイマー及びストップウォッチのカラー

スタート,ストップ
:<shift> + <Enter>

時間セット
:数値入力後<T>

リセット
:<R>

セーブ
:<S>

ロードセーブ	#コメントごとResult.txtに保存
:<L>

データ改ざん
:数値と改ざん内容入力後<X>

データ追加
:追加内容入力後<C>

データ消去
:数値入力後<D>　　#allを入力すると全データを削除

リトライ
:<shift + space>

リディスプレイ
:<Alt> + <R>

シャツpt
:<Q>

バスタオルpt
:<W>

シーツpt
:<E>

減点
:<A>

データ送信
:タイトル入力後<shift> + ボタンクリック
